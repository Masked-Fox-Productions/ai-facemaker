Setup Instructions

To use this demo, you need to configure AWS Cognito:

  1. Create a Cognito User Pool in your AWS account
  2. Create a Cognito Identity Pool with authenticated access
  3. Attach an IAM role with bedrock:InvokeModel permission
  4. Update aws-config.js with your pool IDs

See the README for detailed instructions.
