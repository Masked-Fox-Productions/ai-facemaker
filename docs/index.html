Requirements

To use this demo, you need:

  1. An AWS account with Bedrock model access enabled
  2. IAM credentials with bedrock:InvokeModel permission
How to create IAM credentials
  1. Go to AWS Console ‚Üí IAM ‚Üí Users
  2. Create a new user or select existing
  3. Attach a policy with Bedrock access: 
    {
  "Version": "2012-10-17",
  "Statement": [{
    "Effect": "Allow",
    "Action": "bedrock:InvokeModel",
    "Resource": "*"
  }]
}
  4. Create access keys under Security credentials

See the GitHub repo for more information.
