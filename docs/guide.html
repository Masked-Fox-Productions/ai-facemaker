Generator Guide

Learn how to craft prompts that produce consistent, high-quality character portraits.

The Key Concept: Consistency Through Context

ai-facemaker generates consistent portraits by combining three layers of context:

  1. World Context - The shared universe all characters inhabit
  2. Visual Style - The artistic approach applied to everyone
  3. Character Details - Individual traits for each person

When you keep the world and style consistent, characters from the same project will look like they belong together.

World Context

Describe the setting, time period, and tone of your project. This grounds all characters in the same reality.

Good examples:

Sci-fi game: "Moon Hard Lemonade is a far-future sci-fi cartoon adventure. Humans joined an alien alliance because they love Moon Hard Lemonade. The light side of the Moon is a huge industrial manufacturing hub."

Fantasy novel: "The Ember Kingdoms is a medieval fantasy world where magic flows from volcanic vents. Society is divided between the fire-blessed nobles and the ash-covered workers who tend the forges."

Modern thriller: "A gritty crime drama set in 1970s Chicago. Corrupt cops, desperate criminals, and ordinary people caught in between. Rain-soaked streets and neon-lit diners."

Tips:

  • Include the genre and tone (cartoon, realistic, gritty, whimsical)
  • Mention the time period or technology level
  • Keep it to 2-3 sentences - the model uses this as background context

Visual Style

Define the artistic approach for your portraits. This ensures visual consistency across all characters.

Good examples:

Cartoon style: "Clean, readable sci-fi cartoon portraits. Crisp linework, bold shapes, simple shading. Friendly character design. Consistent cast style. Uncluttered backgrounds."

Painterly style: "Oil painting style portraits with rich, warm colors. Dramatic lighting from a single source. Visible brushstrokes. Renaissance-inspired composition."

Pixel art: "16-bit pixel art character portraits. Limited color palette of 32 colors. Clean pixel placement, no anti-aliasing. Reminiscent of SNES RPG character screens."

Tips:

  • Specify the medium (digital art, oil painting, watercolor, pixel art)
  • Describe lighting preferences (soft, dramatic, flat)
  • Mention color palette tendencies (warm, cool, muted, vibrant)
  • Reference recognizable styles if helpful (Studio Ghibli, noir, Art Deco)

Negative Prompt

List things you want to avoid in the generated images.

Common negatives:

text, watermark, logo, signature, blurry, low-res, cropped, out of frame

Style-specific negatives:

For cartoon styles: photorealistic, photograph, 3D render, uncanny valley

For realistic styles: cartoon, anime, illustration, painting

Character Details

Name

The character's name. This is included in the prompt and helps with consistency if you generate multiple portraits of the same character.

Role

Their job, title, or function in the story. This influences clothing, posture, and expression.

Examples: "Starship Captain", "Village Blacksmith", "Street Urchin", "Corporate Executive", "Rebel Leader"

Description

Physical appearance, clothing, and expression. Be specific about distinguishing features.

Good: "Optimistic smuggler-mechanic, mischievous grin, short messy hair, grease-stained gloves, welding goggles pushed up on forehead."

Too vague: "A woman who works on ships."

Tips for descriptions:

  • Lead with personality/expression - it sets the tone
  • Include 2-3 distinctive visual elements
  • Mention clothing or accessories relevant to their role
  • Keep it concise - long descriptions can confuse the model

Framing

Controls the composition and cropping of the portrait.

Head and shoulders: "Head and shoulders portrait, light background." - Classic bust shot

Close-up: "Extreme close-up face portrait, dramatic lighting." - Emphasizes expression

Three-quarter: "Three-quarter body portrait, environmental background." - Shows more context

Workflow Tips

  1. Lock in your world and style first - Generate a few test portraits until you're happy with the overall look
  2. Save your settings - Copy your world context and style to a text file for reuse
  3. Iterate on characters - Generate 3-4 versions and pick the best one
  4. Use consistent framing - Keep the same framing prompt for all characters in a set
  5. Start cheap, finish expensive - Use Titan for drafts, SD3.5 for finals
