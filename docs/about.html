Consistent Character Portraits at Scale

ai-facemaker is a Python library for generating character portraits that actually look like they belong in the same world.

The Problem

You're building a game, writing a novel, or creating a TTRPG campaign. You need portraits for your characters. You try AI image generation, and each portrait looks like it's from a different artist, a different universe, a different decade.

Consistency is hard. Maintaining a visual style across dozens of characters is even harder.

The Solution

ai-facemaker solves this by separating world context from character details.

You define your world once: the setting, the tone, the visual style. Then for each character, you just describe what makes them unique. The library combines these intelligently to produce portraits that share a consistent aesthetic while still being distinct individuals.

Consistent Style

Define your visual language once. Every character inherits the same artistic approach, color palette, and rendering style.

Multiple Sizes

Generate icons (64x64), bust portraits (256x256), and full artwork (1024x1024) from a single prompt. All sizes use LANCZOS downscaling for crisp results.

Smart Caching

Built-in caching means you never pay twice for the same portrait. Regenerate your entire cast without re-running unchanged characters.

Multiple Models

Choose between Amazon Titan (fast & cheap), Stability SDXL (balanced), or SD3.5 Large (highest quality) based on your needs.

Use Cases

Game Development

Generate consistent NPC portraits for RPGs, visual novels, or strategy games. Export icons for UI and full portraits for dialogue scenes from the same source.

Tabletop RPGs

Create matching portraits for your entire party and important NPCs. Keep visual consistency across a campaign that runs for months.

Fiction & Worldbuilding

Visualize your characters as you write. Build a cast gallery that maintains the tone and style of your world.

Prototyping

Quickly generate placeholder portraits during development. Replace with final art later, or ship with AI-generated portraits for indie projects.

How It Works

from ai_facegen import PortraitClient, WorldSpec, CharacterSpec

# Define your world once
world = WorldSpec(
    context="A dieselpunk noir mystery set in 1940s New Shanghai.",
    style="Moody digital paintings with dramatic shadows and neon accents.",
    negative="blurry, low quality, text, watermark"
)

# Create characters that share the world's aesthetic
detective = CharacterSpec(
    name="Jin Zhao",
    role="Private Detective",
    description="World-weary investigator, rain-soaked trench coat,
                 cybernetic eye glowing amber, cigarette smoke."
)

# Generate portraits
client = PortraitClient(world)
result = client.generate(detective, size=256)
result.save("jin_zhao.png")

Features

  • Python Library - Import and use in your own scripts and pipelines
  • CLI Tool - Generate portraits from JSON config files
  • Batch Generation - Process entire character rosters in one run
  • Intelligent Caching - File-based, memory, or custom cache backends
  • Multiple Output Sizes - 64x64, 256x256, 512x512, 1024x1024
  • High-Quality Downscaling - LANCZOS filtering for crisp small portraits
  • Seeded Generation - Reproducible results with explicit seeds
  • Type Hints - Full typing support for IDE autocompletion
  • Async Support - Non-blocking generation for web applications
